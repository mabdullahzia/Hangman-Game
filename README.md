# Hangman Game

Interactive console-based word-guessing game developed in C++.

## Features
- Random word selection from difficulty-based library
- Tracks guessed letters without repetition
- Input validation for alphabetic entries
- Win/Lose detection and final word display
- Menu-driven interface

## Technologies Used
- C++ Strings, Arrays/Vectors
- Loops, Conditionals, Functions
- Basic DSA techniques for searching letters

## How to Run
1. Open terminal
2. Compile: `g++ main.cpp -o HangmanGame`
3. Run: `./HangmanGame`
